The quickest way to generate documents in large scale. Step by step on how to install and use below.

Installation

To install the package on your computer, you just have to do a simple command on your terminal:

python<version> -m pip install ez-docs

How To Use

In today's version of 'ez-docs', the user needs a markdown template, a database (.csv, .txt or .xls), the pattern of the variables and choose a flag, which will correspond to the type of file the package will create. The value 1 of the flag creates pdf files, while 2 will create markdown files. Opening your terminal, the user should type something similar to:

ez-docs <path to .md template> <path to database> <pattern of variables> --flag=1 or 2
